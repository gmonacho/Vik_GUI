# Vik_Wrap

## Normated

This wrapper respect the cpplint norm. Except "-whitespace/braces", "-readability/braces", "-whitespace/newline".
we favor :
```cpp
funcion()
{
  //  ...
}

if ()
{
  //  ...
}
else
{
  //  ...
}
```
than :

```cpp
funcion() {
  //  ...
}

if () {
  //  ...
} else {
  //  ...
}
```
